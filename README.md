# 📊 Stock Monitor

## Overview

Stock Monitor is a real-time stock tracking tool that integrates live price updates, AI-generated news summaries, and NYSE holiday logic — all powered by GraphQL and intelligent scheduling.


## Features (v0.1.0)

- 📈 Real-time stock price updates (every minute)
- 📰 News fetching every 15–30 minutes
- 🤖 AI summarization of financial news articles
- 👍🏽 Smart pausing of API calls when the market is closed
- 🗓 NYSE holiday integration
- 🎨 Dark/light theme toggle with `next-themes`
- ⚙️ GraphQL API with `graphql-yoga` and `@apollo/client`


## Coming Soon (v0.1.1)

- 💬 **AI Chat Integration:** Chat with the AI agent for deeper market insights directly from summaries.
- 🚨 **Automated Notifications:** Get alerts for significant stock price movements based on user-defined thresholds.
